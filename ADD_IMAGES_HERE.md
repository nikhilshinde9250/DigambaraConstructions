# 📸 How to Add Your Project Images

## Quick Steps:

1. **Find your 6 project images** on your computer

2. **Rename them** to:
   - `1.jpg` (or `1.png`)
   - `2.jpg` (or `2.png`)
   - `3.jpg` (or `3.png`)
   - `4.jpg` (or `4.png`)
   - `5.jpg` (or `5.png`)
   - `6.jpg` (or `6.png`)

3. **Copy all 6 images** to this folder:
   ```
   DigambaraConstructions/public/images/
   ```

4. **Refresh your browser** (if dev server is running)

## Where Images Will Appear:

- **Images 1, 2, 3** → Ongoing Projects page
- **Images 4, 5, 6** → Completed Projects page

## Image Requirements:

- **Format**: JPG or PNG
- **Recommended Size**: 1200 x 800 pixels (or similar 3:2 ratio)
- **File Size**: Under 500KB each (compress if needed)
- **Quality**: High quality but web-optimized

## After Adding Images:

1. Open terminal in `DigambaraConstructions` folder
2. Run: `npm run dev`
3. Visit: http://localhost:5173/projects/ongoing
4. Visit: http://localhost:5173/projects/completed
5. Check if all images display correctly

## Troubleshooting:

**Images not showing?**
- Check file names are exactly: `1.jpg`, `2.jpg`, etc.
- Check they're in the correct folder: `public/images/`
- Refresh your browser (Ctrl+F5 or Cmd+Shift+R)
- Check browser console for errors (F12)
